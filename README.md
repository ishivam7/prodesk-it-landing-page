# Prodesk IT Landing Page

A modern, responsive, and professional landing page developed as part of the "Prodesk IT Sprint-1 Assignment" The website is built using pure HTML, CSS, and JavaScript without any CSS frameworks.


# Project Overview

This project represents a professional IT company landing page that showcases services, company information, and a contact section. It is fully responsive and optimized for desktop, tablet, and mobile devices.


# Features

 Responsive Navigation Bar
 Hero Section
 About Section
 Services Section
 Contact Form
 Professional Footer
 Sticky Navigation Bar
 Dark / Light Mode
 Smooth Scrolling
 Hover Animations
 Mobile Hamburger Menu
 Responsive Design (Mobile, Tablet & Desktop)

# Technologies Used

 HTML5
 CSS3
 JavaScript (ES6)
 Font Awesome
 Google Fonts (Poppins)
